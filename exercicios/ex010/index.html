Usando links

Links externos:

Podes acessar o meu canal de youtube..

Também podes acessar o meu instagram.

Podemos criar links no site com a tag < a >, onde colocamos o link dentro do parâmetro (href="").

Podemos escolher a forma em como este link é aberto (dentro da página, por cima, noutra aba) utilizando o parâmetro (target="") dentro da tag "a".

Ainda dentro da tag "a" podemos descrever ao navegado que tipo de link estamos a coloca, utilizando o parâmetro (rel="").

Links internos:

Esta é a primeira página do site. Se quiseres, podes acessar também a minha segunda página.

Também podes acessar a minha página de notícias.

Para colocar links internos dentro do site, basta dentro do parâmetro "href="(nome da página)" o nome da página para qual quer ser direcionado. Esta página TEM de estar dentro da mesma pasta que a primeira.
