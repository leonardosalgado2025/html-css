Trabalhar com listas

Listas ordenadas:

Estas listas são usadas para items com uma ordem especifica.
  1. Acordar
  2. Ligar mãe
  3. Tomar café
  4. Lavar os dentes
  5. Ir para a facukdade
  6. Almoçar
  7. Ir para casa
  8. Dormir

Para fazer listas ordenadas, utilizamos a tag < ol >, esta ordena os ítems de forma sequencial.
Cada um dos ítems que queremos ordenar devem ser colocados dentro da tag < li >

Podemos ainda alterar o tipo de lista (nomenclação atrás dos ítems) colocando, dentro da tag "ol", o link "type=(lista desejada)".

Adicionalmente, podemos selecionar em que número a lista vai começar.
Para isso utilizamos o lnk "start=(número que queremso começar)".

Listas não ordenadas:

Estas listas são utilizadas quando os items não necessitam de uma ordem especifica.
  • Pão
  • Cerveja
  • Manteiga
  • Tomate
  • Massa

Podemos ainda alterar o tipo de lista (nomenclação atrás dos ítems) colocando, dentro da tag "ol", o link "type=(lista desejada)" .

Listas de definição:

As listas de definição, funciona com um dicionário, ou seja, existe um termo e o seu significado.

HTML
Linguagem de marcação para a criação do conteúdo de um site.

CSS
Linguagem de marcação para a criação do disign de um site.

JavaScript
Linguagem de programaçãp para a criação de iteratividade de um site.

Utilizamos a tag < dl > para a criação de uma lista de definição.
Dento desta utilizamos a tad < dt > para o item e < dd > para o significado.
