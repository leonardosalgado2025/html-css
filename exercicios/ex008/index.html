Principais formatações de texto

Negrito / destaque:

Esta frase contém um negreito com valor semantico de destaque ,
o que deixa a palavra com um tom forte, utilizando a tag < strong >.

Itálico / Ênfase:

Esta frase contém um itálico com valor semntico de ênfase,
utilizando a tag < em >.

Texto marcado:

Podemos criar também um texto marcado, com a Tag < mark >.

E neste parágraafo, temos outro texto marcado.

Texto grande e pequeno:

Criar um texto grande e um texto pequeno.

Texto apagado / inserido:

Podemos marcar um texo como apagado.

Podemos também colo car um texto inserido, para dar uma ênfase e que foi colocado depois.

Texto sobrescrito / subescrito:

Para colocar texto sobrescrito como no caso de (x15+2), usamos a tag < sup >.

Para o texto subrescrito, como no caso de (CO2), usamos a tag < sub >.
